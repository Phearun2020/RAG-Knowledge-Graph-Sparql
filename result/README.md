# Result Folder

Use this folder to keep screenshots and notes from local KG-RAG experiments.

Current examples:

- `Settings.png`: Streamlit settings for local Fuseki + Ollama
- `SPARQL.png`: direct SPARQL query result
- `RAG.png`: RAG query result with generated SPARQL and context

For your master's project, add notes such as:

- Natural language question
- Generated SPARQL
- Returned graph context
- Final answer quality
- KG schema or prompt changes
